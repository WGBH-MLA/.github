## Workflow Templates

### Build
- `build.yml` - Build a docker image and push to a registry
### Format
- `black.yml` - Format Python code with Black
- `lint.yml` - Lint Python code with Ruff
### Test
- `pytest.yml` - Run Python tests with Pytest
- `pytest-with-postgres.yml` - Run Python tests with Pytest and a Postgres database
### Maintainenace
- `update.yml` - Update Python dependencies with PDM
- `sync_labels.yml` - Sync labels across repositories
- `publish.yml` - Publish a release to PyPI
- `docs.yml` - Build and publish documentation to GitHub Pages
- `draft_release.yml` - Draft a release in GitHub
